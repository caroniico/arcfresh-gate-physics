### ⚠️ Known Issues
- **Ice coverage**: SMOS cannot measure SSS through sea ice → NaN

### Usage in Transport
```python
# Salt Flux:  Sm = Σ ρ × (SSS/1000) × v_perp × H × Δx  [kg/s]
# Freshwater: Fw = Σ v_perp × (1 - SSS/S_ref) × H × Δx [m³/s]
# S_ref = 34.8 PSU (Arctic reference salinity)
```

## 4. CMEMS L4 (Gridded SSH/Velocity)

### Source
- **Provider**: Copernicus Marine Service (CMEMS)
- **Product**: `SEALEVEL_GLO_PHY_L4_MY_008_047`
- **Dataset ID**: `cmems_obs-sl_glo_phy-ssh_my_allsat-l4-duacs-0.125deg_P1D`
- **DOI**: [10.48670/moi-00148](https://doi.org/10.48670/moi-00148)
- **Access**: API (`copernicusmarine.open_dataset()`)

### Dimensions

| Dimension | Description | Resolution |
|-----------|-------------|-----------|
| `time` | Daily timestamps | 1 day |
| `latitude` | Latitude grid | 0.125° (~14 km) |
| `longitude` | Longitude grid | 0.125° (~14 km) |

### Variables

| Variable | Dims | Dtype | Units | Description |
|----------|------|-------|-------|-------------|
| **`adt`** | `(time, lat, lon)` | float32 | m | **Absolute Dynamic Topography** — sea surface height above geoid. Primary for DOT. |
| **`sla`** | `(time, lat, lon)` | float32 | m | **Sea Level Anomaly** — deviation from mean sea surface |
| **`ugos`** | `(time, lat, lon)` | float32 | m/s | **Eastward geostrophic velocity** — u-component from SSH gradient |
| **`vgos`** | `(time, lat, lon)` | float32 | m/s | **Northward geostrophic velocity** — v-component from SSH gradient |
| `ugosa` | `(time, lat, lon)` | float32 | m/s | Eastward anomalous geostrophic velocity |
| `vgosa` | `(time, lat, lon)` | float32 | m/s | Northward anomalous geostrophic velocity |
| **`err_sla`** | `(time, lat, lon)` | float32 | m | **SLA formal mapping error** — used for uncertainty propagation |
| **`err_ugosa`** | `(time, lat, lon)` | float32 | m/s | **Eastward velocity formal error** — σ_ugos |
| **`err_vgosa`** | `(time, lat, lon)` | float32 | m/s | **Northward velocity formal error** — σ_vgos |

### Usage in ARCFRESH

```python
# Perpendicular velocity (INTO Arctic):
v_perp = ugos * u_into + vgos * v_into

# Uncertainty propagation:
σ_v_perp = √((σ_ugos · u_into)² + (σ_vgos · v_into)²)

# Volume Transport:
Q(t) = Σ v_perp(x,t) × H(x) × Δx  [Sv = 10⁶ m³/s]
```

### Notes
- **Global coverage**: 1993–present
- **All altimeters merged**: Jason-1/2/3, Sentinel-3, etc.
- **MDT included**: `adt = sla + mdt`, no external geoid needed
- **Daily resolution**: Averaged to monthly for transport calculations

## 11. Transport Formulas

### Volume Transport (VT)
```
Q(t) = Σ v_perp(x,t) × H(x) × Δx    [m³/s → Sv = 10⁶ m³/s]
```

### Freshwater Transport (FWT)
```
Fw(t) = Σ v_perp(x,t) × (1 - S(x,t)/S_ref) × H(x) × Δx    [m³/s → mSv = 10³ m³/s]
```

### Salt Flux (SF)
```
Sm(t) = Σ ρ × (S(x,t)/1000) × v_perp(x,t) × H(x) × Δx    [kg/s → Gg/s = 10⁹ kg/s]
```

### Perpendicular Velocity
```
v_perp = ugos × u_into + vgos × v_into

where (u_into, v_into) = unit vector INTO Arctic (gate-specific)
```

### Uncertainty Propagation
```
σ_v_perp = √((σ_ugos × u_into)² + (σ_vgos × v_into)²)
σ_Q = Σ σ_v_perp(x,t) × H(x) × Δx
```
